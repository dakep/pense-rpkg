# Get the Constant for Consistency for the M-Scale Using the Bisquare Rho Function

Get the Constant for Consistency for the M-Scale Using the Bisquare Rho
Function

## Usage

``` r
.bisquare_consistency_const(delta, eps = sqrt(.Machine$double.eps))
```

## Arguments

- delta:

  desired breakdown point (between 0 and 0.5)

- eps:

  numerical tolerance for equality comparisons

## Value

consistency constant
