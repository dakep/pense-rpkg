# Compute the Tau-Scale of Centered Values

Compute the \\\tau\\-scale without centering the values.

## Usage

``` r
tau_size(x)
```

## Arguments

- x:

  numeric values. Missing values are verbosely ignored.

## Value

the \\\tau\\ estimate of scale of centered values.
